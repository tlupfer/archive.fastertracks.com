FasterTracks Race Tracking

Made with:

Google Maps

Google Maps API provides the map and code to draw the route and markers.

Google App Engine

The site is written in Python, JavaScript, and MySQL. Hosted on Google App Engine.

Spot Tracker and its API

Tracking is done with a SPOT device, which sends the person's location to a satellite every five minutes. Occasionally trees block the transmission, so there is sometimes a delay between GPS pings.

Forecast API

Each time we get a fix on their location, we ask Forecast.io for the weather at that spot.

Twilio

Messages appear after the person sends a text message to a special number, courtesy of Twilio.

Moment.js

We do date formatting with Moment.js

Google Chart

The elevation chart is generated by Google Chart.
